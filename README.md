## Предметная область

Система предназначена для оформления и обработки заявок на определение состава металлического сплава по результатам XRF-анализа: лаборант спектрального анализа вводит описание артефакта и измеренные значения `Cu`, `Zn`, `Sn`, `Pb`, а инженер-спектроскопист проверяет и завершает заявку. Расчёт выполняется как сопоставление измеренного состава с эталонными сплавами: для каждой услуги вычисляется отклонение по элементам `Cu`, `Zn`, `Sn`, `Pb`, после чего выбирается наиболее близкий эталон и формируется итоговый результат совпадения.
